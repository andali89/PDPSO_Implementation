# PDPSO_Implementation
This is an implementation of a probability-based particle swarm optimization algorithm for the product portfolio planning problem.

The source code of this paper will be realease after the acception of the paper. Please refer to the following preprint for the details of the algorithm:

> Liu X, Li A. An improved probability-based discrete particle swarm optimization algorithm for solving the product portfolio planning problem. Research Square; 2022. DOI: 10.21203/rs.3.rs-1860671/v1.
